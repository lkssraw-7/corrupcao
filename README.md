# ATIVIDADE4_THAIS

## Descrição
Este projeto contém análises estatísticas realizadas no notebook `ATIVIDADE4_THAIS.ipynb`. As análises incluem testes de significância, regressões lineares e previsões baseadas em dados de corrupção.

## Estrutura do Projeto
- **ATIVIDADE4_THAIS.ipynb**: Notebook com as análises estatísticas e visualizações.
- **corrupcao.csv**: Base de dados utilizada para as análises da Questão 1.
- **corrupcaoemer.csv**: Base de dados utilizada para as análises da Questão 2.

## Requisitos
- Python 3.8+
- Bibliotecas:
  - pandas
  - seaborn
  - matplotlib
  - statsmodels
  - numpy
  - scipy
  - plotly
  - statstests

## Como Executar
1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Abra o notebook `ATIVIDADE4_THAIS.ipynb` em um ambiente Jupyter Notebook ou JupyterLab.
3. Execute as células sequencialmente para reproduzir as análises.

## Questões Resolvidas
### Questão 1
- Teste F para significância global do modelo.
- Teste t para significância individual das variáveis.
- Equação estimada do modelo.
- Cálculo do R².

### Questão 2
- Comparação de médias entre países emergentes e desenvolvidos.
- Modelo final estimado com variáveis categóricas.
- Previsão para novos dados.
- Intervalo de confiança para as previsões.
